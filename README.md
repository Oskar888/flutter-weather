# Weather App

Created with Flutter.

## Note

This project is still under development

## Getting started
 1. Add your OpenWeatherMap API KEY on lib/main.dart
  ```dart
  static String apiKey = 'YOUR_API_KEY';
  ```
 2. Press GPS button, and wait until the location of the device is displayed
 3. Press API button, after that weather data for the current location is downloaded and displayed

## Features
* Using the location of the device
* Using API for weather data(temperature data and weather icon)
